# 🧠 CareForBrains Website

This repository contains the source code for the official website of CareForBrains.

CareForBrains is a youth-led mental health nonprofit dedicated to promoting mental health awareness, reducing stigma, and expanding access to support through education, advocacy, and community engagement.

---

## 🌐 Live Site

The website is deployed using GitHub Pages:

🔗 [https://careforbrains-stack.github.io/careforbrains-website/](https://careforbrains-stack.github.io/careforbrains-website/)

---

## 📌 About CareForBrains

CareForBrains works to:

* Promote mental health awareness
* Provide education around well-being
* Reduce stigma surrounding mental health
* Support student-led chapters
* Connect volunteers with meaningful opportunities

The website serves as a central hub for programs, chapters, volunteer information, STEM initiatives, e-cards, and ways to connect.

---

## 📂 Repository Structure

```
careforbrains-website/
│
├── images/            # Visual assets and graphics
├── index.html         # Home page
├── about.html         # About page
├── chapters.html      # Chapter information
├── volunteer.html     # Volunteer program page
├── stem.html          # STEM initiative page
├── ecard.html         # Mental health e-cards
├── connect.html       # Contact page
├── logo.png           # Organization logo
└── README.md          # Project documentation
```

All pages are static HTML and are directly deployable via GitHub Pages.
